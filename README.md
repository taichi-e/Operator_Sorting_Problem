# Problem

数字演算子並び換え問題

以下の条件をみたす式で、1、2、3…… という整数をそれぞれ作りたい。
- 3、4、5、6 の 4 つの数字をそれぞれただ一度だけ用いる。
- 演算子には、+、-、*、/ のいずれかを用いる (重複も可)。
- 演算順序を変更する括弧は自由に用いてよい。
 
例えば、
>(4 - 3) / (6 - 5) = 1
>3 - (4 + 6) / 5 = 1
等で 1 が作れる。同様に、
>4 * 5 - 3 * 6 = 2
>(4 - 3) + (6 - 5) = 2
等で 2 が作れる。
 
# (1) 上記のような式を 3、4、5 ……の順番に (可能な限り) 作り、それぞれの式を一つ示せ。
# (2) 上記のような式で作れない、最小の正整数を答えよ。
